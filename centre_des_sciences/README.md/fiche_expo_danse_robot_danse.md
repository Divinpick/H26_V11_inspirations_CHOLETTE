## Description de l'oeuvre

![dispositif](PLUS TARD)

>Photo prise par ZC.

Le dispositif que j'ai choisi de présenter est un petit robot dans lequel tu dois placer des plaque de bois avec des emotions et des actions que le robot pourra effectuer lorsqu'ils sont placer sur les socles
le robot va alors copier les selection que tu as fait et les faires devant toi avec une petite musique.

## Type d'installation

Cette installation est interactive.
